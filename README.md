# 🚀 AI Career Path Mapper

An AI-powered web app that generates personalized career roadmaps based on user input (career goal + current skills).

## 📂 Project Structure

- `app.py`: Main Streamlit app
- `logic/career_logic.py`: Core logic (roadmap + skill gap)
- `data/careers.json`: Career paths and required skills
- `utils/helpers.py`: Helper functions

## 🛠️ Setup Instructions

1. Ensure you have Python installed.
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   streamlit run app.py
   ```
